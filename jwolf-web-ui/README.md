# jwolf-web-ui
```
1.cnpm i -g @vue/cli 或npm i -g @vue/cli,可以需要npm i -g @vue/cli-init
2.vue create jwolf-web-ui,也可以使用IDE或vue ui启动web界面来创建项目
3.进入package.json目录，npm run serve启动，可能报错缺一个module:npm i vue-loader-v16 -S

```

## Project setup
```
npm install
```
